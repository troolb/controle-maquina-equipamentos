# Controle de Máquina e Equipamentos

## Descrição do Projeto
Este projeto é um sistema de controle e monitoramento de máquinas e equipamentos. O objetivo é fornecer uma interface intuitiva para usuários gerenciarem as operações das máquinas, garantindo eficiência e segurança.

## Funcionalidades Principais
- Monitoramento em tempo real das máquinas e equipamentos
- Controle de operação para iniciar/parar máquinas
- Registro histórico de operações
- Alertas e notificações para manutenções e falhas

## Tecnologias Utilizadas
- **Linguagem:** Python
- **Framework:** Flask
- **Banco de Dados:** PostgreSQL
- **Frontend:** HTML, CSS, JavaScript

## Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/troolb/controle-maquina-equipamentos.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd controle-maquina-equipamentos
   ```
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
4. Configure o banco de dados e execute as migrações:
   ```bash
   flask db upgrade
   ```
5. Inicie a aplicação:
   ```bash
   flask run
   ```
